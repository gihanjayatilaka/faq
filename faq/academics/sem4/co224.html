--- layout: QA_page permalink: "/academics/sem4/co224/" parent: "Academics" title: "CO224" author: "" github: faq/academics/sem4/co224.html --- How to run ARM assembly code? Install the following software on your computer (already installed on tesla.ce.pdn.ac.lk) sudo apt-get install gcc-arm-linux-gnueabi sudo apt-get install qemu-user Assemble and run (example is armProg.s) arm-linux-gnueabi-gcc -Wall armProg.s -o assembledCode qemu-arm -L /usr/arm-linux-gnueabi assembledCode
